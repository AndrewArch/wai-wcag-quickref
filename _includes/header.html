[Mockup] How to Meet WCAG 2.0
A customizable quick reference to Web Content Accessibility Guidelines 2.0 requirements (success criteria) and techniques

About

This document provides a customizable list of WCAG 2.0 Success Criteria and Techniques. This allows for focusing on a subset of the guidelines.

Tags & Filters

Tags apply to the Success Criteria. When selecting a tag, only the criteria that are tagged with that tag are visible, the others are collapsed. You can select multiple tags.

Filters can apply to Success Criteria or Techniques.

Note: Other success criteria than the tagged/filtered might also apply in certain circumstances.

Sharing

Use the “Share this view” button to copy the current URL into your clipboard. You can then paste it wherever you like.

Contribute

There are a few ways to contribute to this document. For contributing to the functionality of this quick reference or to report bugs, see this Github repository.

Anyone can also submit techniques that are considered for inclusion in an update. Please submit corrections, updates, or new information related to techniques, failures, or other WCAG documentation to the WCAG Working Group, per the instructions for commenting.
